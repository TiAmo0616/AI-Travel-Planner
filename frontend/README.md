# Frontend

前端（React）说明：

- 目录：`frontend/`
- 主要职责：录音交互、展示行程草案、编辑与导出、与地图服务集成（高德/百度）
- 推荐工具：Vite + React 或 Create React App，UI 框架：Ant Design / Mantine

快速开始（开发环境）：

1. 进入目录并安装依赖

```bash
cd frontend
npm install
```

2. 运行开发服务器

```bash
npm run dev
# 或 npm start（取决于脚手架）
```

后续：在此目录添加 `src/` 源码、`package.json`、`vite.config.js`/`webpack` 配置。
